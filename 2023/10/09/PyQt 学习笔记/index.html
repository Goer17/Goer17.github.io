PyQt 学习笔记

简介

Qt 是一个跨平台的 C++ 开发库，主要用于开发 GUI 程序。Qt 本身是纯 C++ 开发的，但开源社区提供了 Python 等语言的接口。

Qt 支持许多操作系统，例如 Windows、MacOS、Linux 和智能手机的 Android 和 IOS 等等。

PyQt

而 PyQt 是一个用于 Python 编程语言的跨平台 GUI 工具包，它允许开发者创建强大的桌面应用程序和嵌入式应用程序。PyQt 是一个 Python 绑定，提供了对 Qt 库的完整访问。

常用模块

PyQt 有许多功能模块，最常见的模块有以下 3 个：

  • QtCore ：包含了核心的非 GUI 功能，包括一些系统级调用等等；
  • QtGUI ：包含了窗口系统、事件处理、2D 图像、基本绘制、字体文字等等功能；
  • QtWidgets ：包含了一些创建桌面应用的 UI 元素。
控件
按钮系列
  • QPushButton（普通按钮）： 最常见的按钮控件，用于触发各种操作或事件。
  • QCheckBox（复选框）：用于选择一个或多个选项的控件，通常用于启用/禁用功能或选择多个选项。
  • QRadioButton（单选按钮）： 用于从一组选项中选择一个单一选项。通常与其他单选按钮一起使用，以便用户从一组互斥的选项中选择一个。
  • QToolButton（工具按钮）：通常用于工具栏或工具箱中，它们可以包含图标、文本或两者结合，用于执行各种操作。

QLabel

QLabel 是 PyQt5 中的一个标签控件，用于显示文本或图像。它通常用于显示静态文本或图像，并且不接受用户输入。

  • 显示文本或 HTML 内容： QLabel 可以显示纯文本或富文本（HTML）内容。这意味着你可以在标签上显示格式化的文本、链接、图标等。
  • 显示图像： 除了文本，QLabel 也可以用于显示图像，如 PNG、JPEG、SVG 等格式的图像。

QLineEdit

QLineEdit 是 PyQt 中用于创建单行文本输入框的部件，它允许用户在单行中输入文本。这个小部件是很常见的，用于创建各种应用程序中的输入表单、搜索框、用户名密码输入等。
