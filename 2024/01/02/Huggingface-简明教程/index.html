进阶编码

  • return_token_type_ids ：设置为 True 时，会返回一个与每个令牌相对应的序列，用于区分多个句子。例如，在处理句子对（比如问答任务或句子对比任务）时，第一个句子的令牌会被标记为 0，第二个句子的令牌会被标记为 1。
  • return_attention_mask ：当设置为 True 时， 会返回一个 attention_mask。
  • return_special_tokens_mask ：当设置为 True 时，会返回一个 special_tokens_mask。这是一个与输入令牌数量相同长度的数组，用于区分哪些令牌是特殊令牌（如序列开始 [CLS]、序列分隔 [SEP] 等）。特殊令牌被标记为 1，普通令牌被标记为 0。这个信息在后处理步骤中特别有用，比如在需要剔除特殊令牌进行某些计算时。

评价函数

Hugging Face datasets 库提供了一个评价（metrics）功能，用于评估机器学习模型的性能。这些评价函数涵盖了多种常用的性能指标，如准确率、精确率、召回率、F1 分数等，适用于各种任务，包括分类、回归、翻译和摘要等。

  • 加载评价函数：

  • 计算评价指标：

  • 支持的评价指标

    • 分类指标 ：如 accuracy（准确率）、f1（F1 分数）、precision（精确率）、recall（召回率）等
    • 回归指标 ：如 mse（均方误差）、mae（平均绝对误差）等
    • 序列标注指标 ：如 seqeval，适用于命名实体识别等任务
    • 机器翻译指标 ：如 bleu、rouge 等，用于评估翻译质量

管道函数

Hugging Face transformers 库中的 pipeline 函数是一个非常强大且易于使用的工具，它允许我们快速地使用预训练模型来进行各种自然语言处理（NLP）任务，而无需深入了解模型的内部细节。pipeline 封装了模型加载、数据预处理和预测生成等步骤，使得使用复杂的模型变得简单直观。
